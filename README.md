# Kinz ERP - نظام كينز الذكي

نظام متكامل لإدارة الموارد (ERP) مصمم خصيصاً لشركة كينز للأثاث والمطابخ (Kinz Home Art).

## المميزات الرئيسية
- **إدارة العملاء**: تتبع كامل لبيانات العملاء وتواريخهم.
- **إدارة المعاينات**: جدولة ومتابعة المعاينات الفنية (مطابخ/درسينج).
- **إدارة التعاقدات**: إنشاء وتوليد العقود الرسمية بصيغة PDF.
- **المخازن والمشتريات**: تتبع دقيق للمخزون وحركات الصرف والتوريد.
- **المالية والحسابات**: تتبع الحسابات والمدفوعات والتقارير المالية.

## التقنيات المستخدمة
- **React (Vite)**: واجهة المستخدم.
- **Supabase**: قاعدة البيانات والتزامن.
- **Lucide React**: الأيقونات.
- **Glassmorphism Design**: التصميم العصري.

## البدء
1. قم بنسخ مستودع المشروع.
2. قم بتثبيت التبعيات باستخدام `npm install`.
3. قم بإنشاء ملف `.env` وإضافة مفاتيح Supabase.
4. ابدأ التشغيل باستخدام `npm run dev`.
